컴퓨터가 숫자를 표현하는 방법

binary digit = bit 0,1을 순열 방법으로 나타낸다. 즉, 이진법이다. 4=100, 9=1001

비유로 0과1을 전구 on,off로 나타낸다. 각 자리수는 트랜지스터로 나타낸다

8bit = byte

컴퓨터가 문자를 표현하는 방법

A=65, B=66, C=67 -> ASCII코드 : 미국언어 일방적

유니코드: 다른 언어들도 사용가능

다른 해결방법: 이모지(이모티콘) 4036991106 = 기쁨의 눈물 흘리는 이모지 ios: face with tears of joy

색상을 나타내는 방법: RGB
Red degree(0~255) + Green(0~255) + Blue(0~255) 섞으면 여러 색을 표현할 수 있다.

하나의 점(픽셀)에 색상을 담아 여러 점을 모으면 이미지가 된다.

영상: 픽셀이 시간이 지남에 따라 변하는 것 할리우드 영화 24FPS(frame per second) 일련의 사진들

음악: 특정 주파수에 값을 할당

문제해결을 위해서 알고리즘을 인간의 언어로 먼저 적는다

verb = function
condtional = if, else lf
갈림길 = boolean expressions: yes/no true/false 0/1
go back = looop

input=value -> algorithm=function -> output=side effect

추상화: define function 더 읽기 쉽다. 더 생각하기도 쉽다.

변수의 이름은 의미있는 용어를 사용한다.
